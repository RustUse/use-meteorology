### Added

- Added the initial `use-meteorology` workspace with focused primitive vocabulary crates for weather observations, atmosphere, air temperature, atmospheric pressure, humidity, wind, clouds, precipitation, weather fronts, pressure systems, and forecast vocabulary.
- Added the `use-meteorology` facade crate with module aliases and a shared prelude for composing meteorology primitives.
- Added repository baseline files for formatting, linting, local validation, dependency policy, release configuration, and contributor workflows.
