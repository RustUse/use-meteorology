# Maintainers

## Current Release Authority

Until the public maintainer roster is expanded, release authority for RustUse/use-meteorology belongs to the RustUse maintainers with write access to the canonical GitHub repository.

## Responsibilities

Maintainers are expected to:

- review and merge pull requests
- keep the release checklist and changelog current
- publish crates in the documented dependency order
- respond to security and conduct reports in a timely way
- preserve authorship and provenance when porting mirrored contributions

## Updating This File

Changes to maintainer authority should be made by pull request and approved by a current maintainer with canonical release authority.
